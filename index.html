Features

  • Adds small "pips" and "labels" to the slider
  • Extends the jQuery UI Slider widget
  • Easily Configurable
  • Works for both Horizontal and Vertical sliders
  • Doesn't change default slider behaviour
  • Purely a visual plugin
  • IE7+, modern browsers.

Installation

First of all download the /dist/ code from: GitHub and place it into your project structure.

Include jQuery

Make sure you've included a jQuery & jQuery UI package with $.slider
(I've shown them being loaded remotely, depending on your project you may wish to download them locally instead). 

<script src="//code.jquery.com/jquery-1.10.2.min.js"></script>
<script src="//code.jquery.com/ui/1.10.3/jquery-ui.min.js"></script>

Include the plugin

Next we need to include the Pips Plugin. (minified or development)

<script src="js/jquery-ui-slider-pips.min.js"></script>

Include Stylesheets

Make sure you've got the stylesheets loading for the pip plugin.

<link rel="stylesheet" type="text/css" href="css/jquery-ui-slider-pips.css">

Initialize the pips

All that's left now is to initialize the plugin.

At it's most basic you can just run it on a already-initialized slider:

// first we need a slider to work with
var $slider2 = $("#slider2").slider({ max: 20 , value: 10 });

// and then we can apply pips to it!
$slider2.slider("pips");

Initialize the floats

We can also apply floats with default options:

// first we need a slider to work with
var $slider3 = $("#slider3").slider({ max: 20 , value: 10 });

// and then we can apply a float to it!
$slider3.slider("float");

Styling

The default styles are pretty robust, but you can override them yourself by adding your own styles in your own stylesheet. Here's an example of the pips being rounded and placed above the bar instead. I've also shown a different coloured float.

var $awesome = $("#awesome").slider({ max: 20 , value: 10 });
    $awesome.slider("pips").slider("float");
#awesome .ui-slider-pip {
    top: auto;
    bottom: 7px;
}

#awesome .ui-slider-line {
    width: 4px;
    height: 4px;
    border-radius: 2px;
    margin-left: -2px;
    background: #0073EA;
}

#awesome .ui-slider-tip {
    background: #0073EA;
    color: white;
    border-color: #0073EA;
}

#awesome .ui-slider-tip:before,
#awesome .ui-slider-tip:after {
    border-top-color: #0073EA;
}

#awesome .ui-slider-pip-label {
    color: #0073EA;
}

#awesome .ui-slider-label {
    color: #0073EA;
    top: 7px;
}

#awesome .ui-slider-pip-first .ui-slider-label {
    left: -2px;
}

#awesome .ui-slider-pip-last .ui-slider-label {
    left: auto;
    right: -20px;
}

This is 100% css changes with only the default js for pips

Examples

Here is a list of examples with the code used to create them. Combined with CSS the limits are nearly endless, using psuedo selectors like :nth-child() it's possible to show/hide every other pip, every 5th pip or every 10th pip, and the javascript options allow for creating very interesting sliders without muddying your logic!


The default configuration of the pips

$("#slider5").slider({ max: 20 });
$("#slider5").slider("pips");

Here we tell the rest of the points to be label.

$("#slider6").slider({ max: 20 });
$("#slider6").slider("pips" , {
    rest: "label"
});

Here we ask the plugin to hide the rest of the points. Example also shows working with range.

$("#slider7").slider({ max: 20, range: true, values: [ 4, 16 ] });
$("#slider7").slider("pips" , {
    rest: false
});

Here we show pip for the first and last point (no labels).

$("#slider8").slider({ max: 20 });
$("#slider8").slider("pips", { 
    first: "pip", 
    last: "pip"
});

Here we have all label, but crucially we supply a prefix and suffix.
NB: both prefix and suffix are optional and may require slight css tweaks

$("#slider9").slider({ max: 10, value: 5 });
$("#slider9").slider("pips", { 
    rest: 'label',  
    prefix: '$' , 
    suffix: '.00'
});

Here we tell the rest of the points to be label.
We are also assigning custom label for each pip.

var months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'];

$("#slider10").slider({ min: 0, max: 11, value: 5 });

$("#slider10").slider("pips" , {
    rest: 'label', 
    labels: months
})

.on("slidechange", function(e,ui) {
    $("#slider10output").text( "You selected " + months[ui.value] );
});
Nothing Selected.

Here we demonstrate pip and label on a vertical slider with steps.

$("#slider11").slider({ min: 0, max: 100, step: 20, orientation: "vertical" });
$("#slider11").slider("pips");

Vertical Sliders also allow floats and suffix & prefix.

var suffix = "°c"

$("#slider12").slider({ max: 100, step: 20, orientation: "vertical" });
$("#slider12").slider("pips", { suffix: suffix });
$("#slider12").slider("float", { suffix: suffix });

If we've already got pips, we can float those, too. (hover the pips).

$("#slider13").slider({ max: 2 });
$("#slider13").slider("pips");
$("#slider13").slider("float", { pips: true });

API Options

Pips options

Below are the default options for the pip method.
These options can be overidden by supplying an object to the method call.

// Default options for pips:
$slider.slider("pips", {

    first: "label",   
    // string, boolean
    // eg: "label", "pip" , false
    // determines label for first item on slider.

    last: "label",
    // string, boolean
    // eg: "label", "pip" , false
    // determines label for last item on slider.

    rest: "pip",    
    // string, boolean
    // eg: "label", "pip" , false
    // determines labels for all other items on slider.

    labels: false,
    // array
    // eg: ["Mon","Tues","Weds","Thurs","Fri","Sat","Sun"]
    // provides overrides for the labels so the
    // implementer can visually display a different
    // set of labels to the normal value

    prefix: "",     
    // string
    // eg: "$"
    // allows implementer to show a prefix value before
    // the values on the slider

    suffix: ""      
    // string
    // eg: ".00¢"
    // allows implementer to show a suffix value after
    // the values on the slider

});

Float options

Below are the default options for the float method.
These options can be overidden by supplying an object to the method call.

// Default options for floats:
$('div.slider').slider('float', {

    handle: true,
    // boolean
    // eg: true, false
    // activates the floats on slider handles

    labels: true,
    // boolean
    // eg: true, false
    // activates the floats on slider labels

    prefix: "",
    // string
    // eg: "$"
    // allows implementer to show a prefix value before
    // the values on the float

    suffix: ""
    // string
    // eg: ".00¢"
    // allows implementer to show a suffix value after
    // the values on the float

});
